Common HTML Tags

 HTML tag Description
1
 <!DOCTYPE HTML> This document is HTML5 document, It must only appear once, at the top of the page
2
 <html> </html> This root element or beginning of HTML code that is divided into two parts (1) head and (2) body. The visible or display part is body
3
 <head> </head> metainformation about the page. Meta means data about data, that means it will include all the information about and required by the page
4
 <title> </title> there are two usage of title tag (1) in head it is conisder as title of the page shown in chrome or IE tab, (2) in the body it is shown as tool tip text then you define it as an attribute to tag
5
 <body> </body> Visibile container of the HTML page
6
 <h1> --- <h6> These are heading for the page where h1 is maxium and h6 is smallest. There are 6 headings, after closing of tag new line starts
7
 <p> <p> tag specifies the paragraph or text of any length. once <p> tag ends new line starts
8
 <br> the tag used for \n or new line
9
 <b> bold the text it is used inside some tags, after the tag ends there is no new line
10
 <i> italic the text it is used inside some tags, after the tag ends there is no new line
